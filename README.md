# CarWash
Android App.
Разрабатывал приложение для зачета автоматом в команде (остальные занимались сайтом). Не закончил, но автомат поставили:) Скачать apk потыкать можно в релизе

Применяется:
- MVVM
- Hilt
- Retrofit
